
# Project Title

A test script to test the sum of two numbers


## Tech Stack

**Programming language:** Javascript

**Framework:** Jest


## Installation

After cloning the project, install Jest framework with npm

```bash
  npm install --save-dev jest
```
    
## Running Tests

To run tests, run the following command

```bash
  npx jest
```
